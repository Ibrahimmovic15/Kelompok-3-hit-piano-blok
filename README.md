Mini Arcade Game - Hit Piano Block

Deskripsi
Project ini merupakan implementasi game rhythm "Piano Tiles" berbasis Unity yang terintegrasi dengan perangkat mini arcade menggunakan ESP32 sebagai controller.
Game ini mengandalkan kecepatan dan ketepatan pemain dalam menekan tombol sesuai dengan block yang muncul di layar secara sinkron dengan musik.

Tujuan Proyek
Mengembangkan game interaktif berbasis Unity
Mengintegrasikan ESP32 sebagai input controller
Meningkatkan pengalaman bermain melalui sistem scoring dan feedback

Fitur Utama

Sistem lane (jalur vertikal)
Block jatuh mengikuti musik
Sistem hit & miss
Score dan combo system
Feedback visual dan audio

Cara Kerja
1. User menjalankan game
2. Memilih menu mulai
3. Block muncul dari atas layar
4. User menekan tombol sesuai posisi block
5. Sistem mengecek akurasi (hit/miss)
6. Score dan combo bertambah
7. Game selesai jika terlalu banyak miss
